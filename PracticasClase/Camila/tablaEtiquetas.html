MARK La etiqueta mark en HTML se utiliza para resaltar texto dentro de un documento,indicando que es relevante o importante en el contexto actual HTML
SMALL la etiqueta 'small' se utiliza para definir texto en letra más pequeña que el texto normal del documento HTMLHTML
EM la etiqueta 'em' (o etiqueta de énfasis) se utiliza para marcar texto como enfatizado, lo que generalmente se visualiza en cursiva. HTML
CITE La etiqueta HTML cite se utiliza para marcar el título de una obra creativa, como un libro, una canción, una película, etc. No debe usarse para citas de texto o nombres de personas, sino para referenciar la fuente de una cita. HTML
P En HTML, la etiqueta 'p' se utiliza para definir un párrafo de texto. Es un elemento de bloque que crea un espacio antes y después del texto que contiene, separándolo visualmente de otros elementos.

HTML
BLOCKQUOTE La etiqueta 'blockquote' en HTML se utiliza para indicar una sección de texto que es una cita de otra fuente. Esta etiqueta ayuda a diferenciar visualmente la cita del resto del contenido de la página, a menudo mediante sangría. Es un elemento de bloque, lo que significa que ocupa todo el ancho de su contenedor.
HTML
ABBR La etiqueta HTML abbr se utiliza para definir una abreviatura o acrónimo en un documento HTML. El atributo title se usa para proporcionar la descripción completa de la abreviatura, que se muestra al pasar el cursor sobre el texto. HTML
CODE La etiqueta code en HTML se utiliza para mostrar texto como código de computadora. Este texto, dentro de la etiqueta, no se interpreta como HTML, sino que se muestra tal cual, generalmente con un formato de fuente monoespaciada para distinguirlo del resto del contenido. SUPERINDICE Y SUBINDICE En HTML, las etiquetas 'sub' y 'sup' se utilizan para mostrar texto como subíndice y superíndice, respectivamente. El subíndice se muestra ligeramente debajo de la línea base, mientras que el superíndice se eleva por encima. Estas etiquetas son útiles para expresiones matemáticas, fórmulas químicas, notas al pie y otras situaciones donde se requiere un desplazamiento vertical del texto. 52,52
TACHADO Y SUBRAYADO En HTML, el tachado de texto se logra principalmente con las etiquetas 's' o 'del'. Ambas etiquetas producen un resultado visual similar, pero tienen diferentes significados semánticos. La etiqueta 'del' se utiliza para indicar que el texto ha sido eliminado o ya no es válido, mientras que 's' simplemente aplica un tachado visual. HTMLHTML
STRONG La etiqueta strong en HTML se utiliza para indicar que el texto contenido dentro de ella tiene mayor importancia o relevancia dentro del contenido. Aunque a menudo se muestra en negrita, su función principal es semántica, señalando la importancia del texto, no solo su apariencia visual. HTML
B La etiqueta b en HTML se utiliza para hacer que el texto aparezca en negrita, sin agregar ninguna importancia semántica. Es una etiqueta de formato que simplemente cambia la apariencia visual del texto. HTML
I La etiqueta i en HTML se utiliza para definir una parte del texto con un estilo alternativo, generalmente cursiva, y a menudo para resaltar palabras o frases que tienen un significado especial o son términos técnicos. Aunque en versiones anteriores de HTML, i se usaba simplemente para aplicar cursiva, en HTML5 su uso se ha refinado para representar un tono de voz o estilo diferente, sin necesariamente enfatizar la importancia del texto. HTML
U En HTML, la etiqueta u se utiliza para subrayar texto. Aunque originalmente se usaba para indicar texto que no era parte del contenido principal o para resaltar errores tipográficos, su uso ha disminuido en favor de CSS para el subrayado visual. HTML
UL En HTML, la etiqueta ul define una lista desordenada, también conocida como lista con viñetas. Se utiliza para agrupar elementos que no tienen un orden específico. Cada elemento de la lista se define con la etiqueta li dentro de la etiqueta ul.

LI En HTML, la etiqueta li (de "list item") se utiliza para definir elementos individuales dentro de una lista. Estas listas pueden ser ordenadas (con ol) o desordenadas (con ul). Cada elemento li representa un ítem en la lista.
